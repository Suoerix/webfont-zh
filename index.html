字体子集化分包服务

📋 API 接口

GET获取字体列表

/api/v1/list

返回所有可用字体的配置信息，包括字体ID、版本、许可证等。

测试接口

GET获取字体子集

/api/v1/font?id={font-id}&char={codepoints}
id: 字体ID (必需) char: Unicode十进制码点，逗号分隔 (必需)

返回包含指定字符的WOFF2字体文件，自动缓存以提高性能。

测试"中"字 测试生僻字

POST重新生成字体

/api/v1/generate?id={font-id}&char={codepoints}
id: 字体ID (可选) char: Unicode十进制码点 (必需)

强制刷新字体缓存文件，如果无缓存则直接生成字体文件。
